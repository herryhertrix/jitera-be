<h2>🛠️ Installation Steps:</h2>

<p>1. install all library</p>

```
yarn install
```

<p>2. create .env.development and copy from .env.development.example</p>

```
.env.development
```

<p>3. This BE will use port 3030</p>

```
localhost:3030
```

<p>4. start project</p>

```
yarn start
```
